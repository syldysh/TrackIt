<h1 align="center">TrackIt</h1>

<p align="center">
Мобильное iOS-приложение для управления задачами и планирования времени.
</p>

Проект построен на архитектурном паттерне <strong>MVVM (Model-View-ViewModel) с выделенным Domain Layer</strong>, адаптированном под iOS-приложения.
Подробнее об этом подходе можно прочитать в статье на <a href="https://habr.com/ru/articles/808923/" target="_blank">Habr</a>. <br><br>
Дизайн приложения (первоначальный интерактивный прототип): <a href="https://www.figma.com/make/Aye5SHZt3qegU7fLjJGpv6/High-Fidelity-UI-Design?fullscreen=1&t=ntohwQQQznTi8c4d-1&code-node-id=0-9" target="_blank">Figma</a> <br>
